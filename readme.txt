A lightweight rental car inventory and booking agreement system for WordPress.

== Description ==

Rental Car Agreements is a simple, effective plugin for managing vehicle rentals. It allows you to:
* Manage a fleet of vehicles (CPT).
* Receive booking requests via a frontend form.
* Manage bookings in the backend.
* Automatically generate printable rental agreements.

**Features:**
* Vehicle Inventory Management.
* Booking Request System.
* Frontend Shortcodes (`[rental_car_inventory]`, `[rental_car_booking]`).
* Custom Settings for Business Info.
* Printable PDF-ready Agreements.

== Installation ==

1. Upload the `rental-car-agreements` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "Rental Car Settings" to configure your business information.
4. Add vehicles under "Vehicles".
5. Create a page with `[rental_car_inventory]` to show your cars.

== Shortcodes ==

* `[rental_car_inventory]` - Displays grid of available vehicles.
* `[rental_car_booking]` - Displays the booking form. Accepts `vehicle_id` parameter.
